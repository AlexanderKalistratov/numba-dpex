Extending Numba

Extending Numba describes how to extend Numba to make it recognize and support additional operations, functions or types.

Numba provides two categories of APIs:

  • The high-level APIs

  • The low-level APIs

Example:

1@numba.njit
2def sum(a, b):
3  return a + b
4
5a = dpnp.ndarray([10])
6b = dpnp.ndarray([10])
7c = sum(a, b)
