numba_dppy.testing module¶

numba_dppy.testing.assert_auto_offloading(parfor_offloaded=1, parfor_offloaded_failure=0)¶

If parfor_offloaded is not provided this context_manager will check for 1 occurrance of success message. Developers can always specify how many parfor offload success message is expected. If parfor_offloaded_failure is not provided the default behavior is to expect 0 failure message, in other words, we expect all parfors present in the code to be successfully offloaded to GPU.
